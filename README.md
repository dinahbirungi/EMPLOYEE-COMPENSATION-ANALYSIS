## 1. Overview
This project focuses on analyzing workforce compensation data to understand salary distribution, benefits structure, overtime costs, and employee composition across departments and job roles.

Using SQL and Power BI, the project transforms raw employee compensation data into meaningful insights that support better workforce planning, budgeting, and compensation decisions.

The final output is an interactive Power BI dashboard that allows users to explore compensation trends by department, job, and employment type.

---

## 2. Objectives
The key objectives of this analysis are to:
- Analyze average and total compensation across departments
- Identify job roles with the highest total compensation
- Examine overtime cost distribution by department
- Understand employee distribution by employment type
- Provide clear visuals to support data-driven decision-making

## 4. Data Preparation
The dataset was prepared using the following steps:
- Reviewed and cleaned raw compensation data
- Ensured correct data types for salary, overtime, and benefits columns
- Verified calculations for total salary, total benefits, and total compensation
- Removed inconsistencies and validated missing values
- Structured the data for efficient analysis and visualization

---

## 5. Techniques Used
- Aggregation using `SUM`, `AVG`, and `COUNT`
- Grouping data by department, job, and employment type
- KPI calculations for average salary and total compensation
- Ranking to identify top-paying jobs and departments
- Interactive filtering using slicers in Power BI
- Data storytelling through charts and dashboards

---

## 6. Recommendations
Based on the analysis, the following actions are recommended:
- Monitor departments with high overtime costs to control payroll spending
- Review compensation structure for high-paying job roles to ensure alignment with organizational goals
- Use workforce distribution insights to support staffing and budgeting decisions
- Leverage dashboards for regular compensation reviews and workforce planning
